Catacaustic Curve Generator

School Project

Java  ·   JavaFX
Memory Optimization  ·  Run-Length Encoding  ·   File I/O  ·  Complexity Theory

Objective:

Create an interactive curve generator that can animate how changing parameters affect a curve.Create a program generates catacaustic curves using times tables.

Program Design:

A set of points are indexed and placed around the edge of a circle. Two points are chosen by multiplying the current index by the multiplier. A line is then drawn between the two points and index is increment. This process is repeated many times until the curve appears. The user can choose the number of points and the multiplier. In addition, these values can be changed over time resulting in an animated effect.

What I Learned:

How to use JavaFX tools to animate graphical elements, and safely change variables from different threads.

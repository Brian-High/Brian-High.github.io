Professional Application - Ideum

Unity  ·   C#/.NET
Micro-interactions  ·  Animations  ·   Content Management System  ·  Attention To Details

Objective:

Create and educational and interactive program that runs on a touchscreen and wall mount display. This application was a remake of an existing exhibit at the Fort Worth Zoo from the 80s. The goal was to give the application a cleaner design and a more interactive feel.

Design:

The original exhibit used an array of physical buttons, one for each animal's habit during 3 times in history. The new program uses a slider to control the timeline in conjunction with 7 buttons to control the which animal is displayed. Since this program is so simple a lot of time was spent designing micro-interactions and animations to give it a polished feel.

The Solution:

My solution was to use the DOTween unity plugin which is more suited to the of controlling the execution of the program. Also I converted all the 4k videos to hd to increase the performance of the application. In the end, the new version of the application was identical to the original with some small improvements and functioned perfectly.

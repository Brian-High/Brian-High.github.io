Professional Application - Ideum

Unity  ·   C#/.NET
Micro-interactions  ·  Animations  ·   Content Management System  ·  Attention To Details

Objective:

Create and educational and interactive program that runs on a touchscreen and wall mount display. This application was a remake of an existing exhibit at the Fort Worth Zoo from the 80s. The goal was to give the application a cleaner design, more intuitive interface, and a more interactive feel.

Design:

The original exhibit used an array of physical buttons, one for each animal's habit during 3 times in history. The new program uses a slider to control the timeline in conjunction with 7 buttons to control the which animal is displayed. Since this program is so simple a lot of time was spent designing micro-interactions and animations to give it a polished feel.

Content Management System (CMS)

The program was designed flexibly to allow for client to change the content of the app after it is deployed. The Unity build has an exposed folder called Streaming Assets which is used to load the content. Due to the simplicity of this program, the CMS is relatively simple. The content can be changed by replacing the desired file and changing the name to original files .
